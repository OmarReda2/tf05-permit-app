Fix Slice 10 dashboard/permit filtering and expiry labeling issues only.

Current issues:
1. In the permit list, filtering by Approved and Rejected works.
2. Filtering by Expired does not show permits, although some permits display expiry state as Expired.
3. Empty state shows "No permits found. Create your first permit." for HSE/CM users, but they are not authorized to create permits.
4. Pending permits show expiry state as "Not Active", which is confusing.

Root rule:
Do not confuse workflow status with expiry state.

Definitions:
- Workflow status is stored in Firestore: SUBMITTED, HSE_APPROVED, APPROVED, REJECTED, CLOSED.
- Expiry state is calculated in UI from status + expiryTime.
- Firestore status should not be changed to EXPIRED in this slice.

Required behavior:
- Status filter should filter workflow status only.
- Expiry filter should filter calculated expiry state only.
- If there is currently one mixed filter, split it into:
  1. Status filter
  2. Expiry state filter
- Expired filter must use calculated expiry state, not Firestore status == EXPIRED.
- Approved permits can be Active, Expiring Soon, or Expired based on expiryTime.
- Pending HSE, Pending CM, Rejected, Closed should show expiry state as "Not Applicable" or "Awaiting approval", not "Not Active".

Preferred expiry labels:
- APPROVED + future expiry > 2 hours: Active
- APPROVED + future expiry <= 2 hours: Expiring Soon
- APPROVED + expiry in the past: Expired
- SUBMITTED: Awaiting HSE Approval
- HSE_APPROVED: Awaiting Final Approval
- REJECTED: Not Applicable
- CLOSED: Closed

Empty state behavior:
- SITE_USER with no permits may see: "No permits found. Create your first permit." with link to /permits/new.
- ADMIN/HSE_MANAGER/CONSTRUCTION_MANAGER should not see a create-permit link.
- For non-site users, show: "No permits found for the selected filters."

- Physically update permit status to EXPIRED.
- Add Cloud Functions.
- Add dashboard charts.
- Add approval logic.
- Add PDF/export.
- Add dependencies.
- Refactor unrelated auth/admin/approval code.
